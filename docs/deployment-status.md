## All three readers run the mechanical-first pipeline (deploy-then-refine)

Every live edition is **mechanical-first method output** (v0 → v1 clause-atoms → v1.5 binding rules), deployed and then refined. The hand-edited editions were **superseded** — they are NOT a protected "sacred" layer and NOT a deploy oracle. The deploy bar is **"new AND superior by the method's own bidirectional-test yardstick,"** never "as good as the old hand-edits." (See `feedback_hand_edit_is_a_datapoint`.)

**What *is* still sacred:** the **source text** — v0 word forms, consonants/niqqud/te'amim (Hebrew), punctuation, verse refs. Never modified. The ATU **line-breaks/segmentation** are regenerable method output, not hand-edits to protect.
