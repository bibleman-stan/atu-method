# Scholarship — Universal Methodology Essays

This directory holds cross-cutting methodology essays — the defensibility, intellectual lineage, and empirical-validation arguments for the apparatus as a whole.

Audience: the scholar — peer reviewer, dissertation reader, future researcher, future-Stan validating the framework's defensibility against challenges.

Companion to: [`../../docs/framework.md`](../../docs/framework.md) (the operational framework specification). The framework doc contains WHAT (the methodology's normative requirements). This directory's essays contain WHY (defensibility, lineage, empirical grounding).

## When this content is published

These essays are the publication-ready material for a methodology paper (or dissertation chapter) about the apparatus. They evolve slowly, with deliberate care. Updates require the same audit discipline as canon changes (§7.3 trigger #1 if a new methodology essay is added; lower triggers for edits within existing essays).
