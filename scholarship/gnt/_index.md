# Scholarship — GNT Per-Rule Companions

This directory holds rationale, grammatical grounding, empirical evidence, and intellectual lineage for each rule in the readers-gnt operational canon (`readers-gnt/private/01-method/colometry-canon.md §3`).

Audience: the scholar — peer reviewer, dissertation reader, future researcher applying the apparatus to a new corpus, or future-Stan validating defensibility.

Companion to: each rule entry in the GNT operational canon. The operational canon entry contains WHAT/HOW (rule statement, UD signature, scope, exclusions, examples, implementation refs). This directory's per-rule files contain WHY/HOW WE KNOW (rationale, defensibility, empirical evidence, adversarial history).

## Cross-references

When a per-rule scholarship file cites a universal cross-cutting concept (M1 bonded pair, N=2 adjudication principle, audit-extension trigger), it links to the universal essay in [`../methodology/`](../methodology/) rather than re-stating.

When a per-rule scholarship file cites another GNT rule, it uses stable rule ID (R10, R19, R20, etc.). Section-position citations are forbidden.

The operational canon entry for each rule lives in `readers-gnt/private/01-method/colometry-canon.md §3` and is the single source of truth for WHAT/HOW. The scholarship companions in this directory say WHY/HOW WE KNOW.
