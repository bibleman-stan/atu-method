# Scholarship — BoFM Per-Rule Companions

This directory holds rationale, grammatical grounding, empirical evidence, and intellectual lineage for each rule in the readers-bofm operational canon (`readers-bofm/private/01-method/colometry-canon.md` §5).

Audience: the scholar — peer reviewer, dissertation reader, future researcher applying the apparatus to a new corpus, or future-Stan validating defensibility.

Companion to: each rule entry in the BoFM operational canon. The operational canon entry contains WHAT/HOW (rule statement, UD signature, scope, exclusions, examples, implementation refs). This directory's per-rule files contain WHY/HOW WE KNOW (rationale, defensibility, empirical evidence, adversarial history).

## Cross-references

When a per-rule scholarship file cites a universal cross-cutting concept (M1 bonded pair, N=2 adjudication principle, audit-extension trigger), it links to the universal essay in [`../methodology/`](../methodology/) rather than re-stating.

When a per-rule scholarship file cites another BoFM rule, it uses stable rule ID (R7, R17, R26, etc.). Section-position citations are forbidden.
