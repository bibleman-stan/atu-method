3. **Same asymmetry as grammar.** Per [[grammar-constrains-not-determines]]: grammar gives PROHIBITIONS, atomic-thought gives PRESCRIPTIONS. **How to apply:**

1. **When auditing a Stan-flagged verse with apparent rhetorical structure:** identify the figure (hendiadys / merism / parallelism / chiasm / etc.), note its default disposition, then RUN THE BIDIRECTIONAL ATOMIC-THOUGHT TEST. 2. **When the figure's default and the atomic-thought test agree:** apply the agreed disposition. The figure's default helps predict; the test confirms.
