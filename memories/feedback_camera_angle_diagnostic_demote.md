**Principle.** The "camera-angle shift" diagnostic — described in framework §1.3 as a TIEBREAKER alongside single-image (and listed in §1.6 precedence tier-matrix as DIAGNOSTIC) — has been used in session to JUSTIFY splits that would otherwise fail the atomic-thought test. Stan's verbatim 2026-05-13: *"the 'camera angle change' should probably either drop out or be caveated as 'when it is its own complete thought' — the more I see it at work, the less confident I am it's helpful."*

When examined under pressure, camera-angle reasoning either:
- (a) **Collapses to the atomic-thought test** — "this side has its own atomic content; that side has its own atomic content" is exactly what the §1.1 generative principle (now sharpened by bidirectional refinement per [[atu-test-is-bidirectional]]) tests for. Camera-angle adds no independent diagnostic force; it's a paraphrase.
- (b) **Smuggles in aesthetic/rhetorical preference** — "the rhythm wants a beat here," "the visual cadence is improved," "the narrative emphasis is sharper." This is [[rhetoric-bandwagon]] failure mode. Aesthetic force without atomic-thought-independence is exactly what [[rhetorical-force]] rejects.

**Why:** The diagnostic was supposed to function as a tiebreaker WHEN the atomic-thought test was genuinely indeterminate. In practice, when invoked, it has been functioning either redundantly (collapsing to the test) or as cover for aesthetic-judgment-as-mechanical-determination. The asymmetry from [[grammar-constrains-not-determines]] applies: camera-angle isn't a determination engine. Stan's session-cumulative observation: the diagnostic isn't helpful enough at its original weight; the bidirectional ATU test now does the work that camera-angle was sometimes called in to do.

**Worked example confirming the demote (2026-05-13 BoFM Alma 31:11):** YES/NO argument on *"to preach the word unto them"* as standalone ATU. The YES side leaned on camera-angle shift (motion-event vs preaching-event). The NO side leaned on bidirectional ATU test failure (forward fragment + backward anaphoric *them*) plus canon R7 SCOPE. The NO side carried. Stan: *"i agree the no argument is stronger; as such, you know what to do, right?"* — confirming camera-angle was the weaker channel. The R7 SCOPE-merge wave applied 73 corpus merges with the camera-angle-as-justification-defense correctly rejected.

**How to apply:**

1. **When tempted to argue "camera-angle shift justifies a split":** STOP. Re-run the atomic-thought test in BOTH directions (per [[atu-test-is-bidirectional]]). If atomic-thought-test fails in either direction → MERGE (camera-angle doesn't rescue it). If atomic-thought-test passes in both directions → camera-angle adds nothing; the test already settled it. Camera-angle is never the determinant.

2. **When auditing a Stan-flagged verse:** treat appearance of camera-angle reasoning in the analysis as a warning signal. If a YES-argument (for keeping split) leans on camera-angle to defend the existing break, the YES-argument is likely the weaker side. The bidirectional atomic-thought test (or named R/M/EP rule) should be doing the work.

3. **When proposing new rule extensions:** never use camera-angle as the justification. The atomic-thought test (bidirectional) is the canon-grounded determination engine; grammar constrains it; rhetoric-figures constrain it ([[rhetoric-figures-constrain-atu]]); camera-angle does not add diagnostic force.

4. **Framework-edit (this memory's load-bearing artifact):** framework §1.3 should be updated to caveat camera-angle as "deprecated; redundant with bidirectional atomic-thought test under §1.1 refinement (see [[atu-test-is-bidirectional]]). Invocation is a warning signal for bandwagon/aesthetic reasoning." §1.6 precedence tier-matrix should likewise demote image-diagnostic row or annotate it as deprecated.
