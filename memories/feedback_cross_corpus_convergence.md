# Cross-corpus ATU convergence — hypothesis, and the discipline that earns it

**Stan's program thesis (verbatim sense, 2026-05-20):** *ATUs reveal common patterns/solutions across languages, because that is how thought and language work.* Building out multiple corpora (Tanakh → GNT → LXX → Vulgate → rebuilt BoM) should EMPIRICALLY DEMONSTRATE this convergence; and each demonstrated solution lets the next reader be built faster by replicating what already worked.

**Observed instance (the one that prompted this):** the Greek binding-rule catalog turned out to mirror the Hebrew B1–B14 catalog rule-for-rule — Greek object/indirect-ὅτι ↔ Hebrew **B11** (verb-of-cognition + ki-complement); recitative-ὅτι = the speech-verb inverse of B11; restrictive ὅς-relative ↔ **B3** (restrictive ʾăšer binds); interjected vocative ↔ **B1**; complement/purpose infinitive ↔ **B10**; attributive participle ↔ **B13**. ὅτι *is* ki; ὅς *is* ʾăšer, at the clause-binding level.

**The discipline (this is the load-bearing part):**
1. **Look back first.** Before treating a new-corpus problem as novel, find its analog in an already-built corpus (Tanakh is the deepest-built). Port the proven mechanism. See the consumer-side habit in the user-home memory `feedback_check_prior_corpora`.
2. **But EARN the convergence — do not assume it.** Porting a Hebrew rule and declaring "Greek matches" is circular and would fake the very evidence the program wants. Each ported rule must independently pass the bidirectional atomic-thought test ([[feedback_atu_test_is_bidirectional]]) and the rule-proposal gates ([[feedback_rule_proposal_gates]]) **on its own corpus's data**. The convergence is only evidence if the Greek rule was confirmed against Greek cold-eye independently, then *observed* to coincide with the Hebrew one. Counterexamples are signal, not noise (e.g. Matt 2:6 `ὅστις ποιμανεῖ` binds despite ὅστις — the B3-analog needs restrictiveness, not a naïve particle check, exactly as Hebrew B3 did).
3. **Scope-stake nothing prematurely.** Per staged-paper discipline, the convergence is a HYPOTHESIS the multi-corpus build tests — not an asserted result to headline. Let the catalogs accumulate; the cross-linguistic claim is earned by the corpora, in a later stage, not announced early.

**Why it matters:** convergence that is observed (not assumed) is the strongest cross-corpus result the methodology can produce, AND it is the practical engine that accelerates LXX / Vulgate / BoM. Both payoffs require the same move: replicate the proven solution, then independently re-derive it on the new corpus so the match is real.

**The three-layer model (Stan + Claude, 2026-05-20) — language, genre, figure:**
1. **The ATU is the universal invariant** — a thought is a thought, language- and genre-independent; the bidirectional test never moves.
2. **Language constrains the RULE CATALOG** that realizes ATUs (Greek ὅτι/participle/gen-abs vs Hebrew כי/wayyiqtol/NmCl). This is the convergence axis.
3. **Genre is DIAGNOSTIC, not prescriptive** — it is the *distribution/sampling* layer, NOT a peer rule-catalog of language. We never write "genre rules." We apply the one unified method and the method *outputs* genre structure: gaps surface exactly where a genre clusters constructions the catalog hasn't covered. The fix is always a genre-AGNOSTIC language-catalog rule keyed to a grammatical signature (e.g. the Greek verbless-vision-clause gap apocalyptic surfaced is filled by porting Hebrew NmCl — a Greek rule usable in any genre). **Genre-spread is the instrument that proves the language catalog is complete** (hence the 4-genre Hebrew / 3-genre Greek validation designs). The genre-surfaced gap is filled by cross-LANGUAGE convergence: genre reveals, convergence fills — the two axes interlock.
4. **Figures (parallelism, hendiadys, chiasm) have ATU dispositions that are themselves cross-linguistic** (parallelism→split members in both Hebrew & Greek), genre-correlated only in *density* (hymns/poetry). So even figures are not genre rules. See [[feedback_rhetoric_figures_constrain_atu]].

**"Overcame" is honest, not absolute:** the unified catalog produces a refinable DRAFT across all genres; the **editorial-absorption rate varies by genre** (poetry/apocalyptic > narrative). Tanakh proved this (B1–B14, one construction-based catalog, validated across Gen22/Ps1/Isa53/Lev11 with NO genre-specific rules). GNT is the second run of the same move.
